# 대학생, 사회 초년생을 위한 루틴 관리 앱 <머니핏>

📱 머니핏은 소비 습관을 체계적으로 관리하고, 올바른 금융 습관을 형성하도록 돕는 모바일 앱입니다.
20대를 위한 맞춤형 루틴 관리와 AI 기반 소비 분석을 통해, 개인과 그룹 모두의 소비 습관을 개선하는 것을 목표로 합니다.
---

## ✨ 주요 기능

### 1. 소비 루틴 관리

#### 계좌 연동 및 AI 소비 패턴 분석
- 사용자의 은행 계좌와 연동하여 소비 데이터를 수집  
- AI가 자동으로 소비 패턴을 분석  

#### 소비 피드백 및 루틴 추천
- 분석 결과 기반 절약 포인트, 예상 절약 금액 제공  
- 생활 속 실천 가능한 루틴 추천  
  - 예: "주 3회 집밥 먹기", "택시 대신 버스 이용"  

#### 맞춤 금융 상품 추천
- 사용자의 신용 정보, 예적금 현황, 성향 등등 분석  
- 예·적금 상품 추천  
- 이자 부담을 줄일 수 있는 적절한 대출 상품 제안  

---

### 2. 단체 루틴 관리

#### 단체 루틴 구성 방식
- **커뮤니티 기반 모임**: 먼저 팀을 구성 → 루틴 결정  
- **루틴 기반 모임**: 루틴 선택 → 같은 시간·날짜에 참여할 사람들과 팀 구성  

#### 단체 방 운영 기능
- 단체 방 개설 → 채팅 기능 제공  
- FCM(Firebase Cloud Messaging) 연동 → 알림 발송  
- 커뮤니티 형식으로 단체 루틴 모임 구성 기능으로 단체 루틴 관리
- 절약 챌린지 등 성취 공유  

---

### 3. 루틴 생성 기능

#### AI 추천 루틴 제공 
- 사용자의 소비 패턴과 유사한 타인의 데이터를 기반으로 맞춤 루틴 생성 

#### 사용자 직접 루틴 생성
- 원하는 카테고리와 내용용을 설정하여 자유롭게 루틴을 설계

---

### 🛠️ 사용 기술

- FE: React Native, TypeScript, React Query 
- BE: Spring Boot, Java
- DB: MySQL, Redis
- Infra: Docker, Nginx
- AI
1. KNN 기반 추천 시스템
 - 다른 사용자와의 비교를 통해 더욱 적합한 루틴 소개
2. KoBERT를 이용한 소비 내역 분석
 - 계좌 연동 후 입출금 내역 파악, 사용 내역 별로 소비 카테고리 분류
3. Light GBM Ranker
 - 사용자의 소비 내역 분석 후, 그에 적합한 금융 상품 소개
